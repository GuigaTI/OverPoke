# OverPoke

Overlay estilo Pokémon com contador e GIFs, criado em Python (PyQt6).

## Funcionalidades

- Overlay transparente com GIF animado.
- Contador estilo pixel atualizado por hotkey.
- Redimensionável e movível.
- Vários temas (FireRed, Emerald, Yellow, Minimal).
- Lock/Unlock de overlays e hotkey configurável.
- System tray para acesso rápido.

## Como usar

1. Instale Python 3.11+.
2. Instale dependências:
   ```bash
   pip install PyQt6 keyboard
3. Execute o Script:
   python overpoke.py
4. Ou use o .exe
